This Online Manchu Dictionary is a small web-app that contains a dictionary of the Manchu language. The Manchu language is an almost extinct language in the world today, with only a few tens of first-language speakers of it left. This web-app is being developed gradually over time, so more data will appear. At the moment, it has {{$scope.entries.length}} entries.

You can see the source code for this project on GitHub: https://github.com/BenjaminTMilnes/ManchuDictionary. This project has been created by B. T. Milnes

Typing Manchu text can be difficult. Try using this tool, which will take romanised Manchu, and output Manchu script.
